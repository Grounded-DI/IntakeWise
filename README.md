# IntakeWise DI2 — Intake QC / Validation Demo

**Prepared by:** Grounded DI LLC  
**Demo Type:** Intake QC / Validation  
**Use Case:** Commercial property water-loss / mold exposure intake  

IntakeWise DI2 is a deterministic intake-review workflow support module designed to help reviewers identify contradictions, unsupported statements, missing information, escalation triggers, and structured export needs across messy intake records.

This demo converts a multi-source intake packet into a source-linked chronology, contradiction matrix, missing-information register, escalation recommendation, confidence routing summary, and audit capsule.

The system is not presented as a replacement for human review or as a legal / coverage decision-maker. Its purpose is narrower: to make intake risk visible earlier, improve review consistency, and create an audit-ready packet for claims, legal, compliance, or operations teams.
